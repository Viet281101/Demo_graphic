### Installation

- Install [GL4Dummies](https://github.com/noalien/GL4Dummies)
- Install [SDL2](https://wiki.libsdl.org/SDL2/Installation) :

```
$ sudo apt-get install libsdl2-dev
```


### Compilation

- Utiliser le **_Makefile_** pour compiler le projet:

```
$ make && ./demo
```
